# Caafi-Hub Client

React.js frontend for the Caafi-Hub Water Supply & Fleet Management System.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

Make sure `caafi-hub-server` is running on port 5000 first.
